# vue2-sample-project

## Used Modules

The used modules except the core vuex modules in the project

- bootstrap (style)
- bootstrap-vue (style)
- sass (style)
- sass-loader (style)
- vuex (state management)
- vue-i18n (multilanguage)
- vue-router (routing)
- vue-select (searchable select input)

## Project setup

```
npm install
```
